           1. creat a project                     :ng new admin-dashboard

           2.run project                          :ng serve(inside angular project)

           3.import or add 3rd party libraries    :     -install angular material(ng add @angular/material)
                                                        -choose a prebuild theme
                                                        -enable typography
                                                        -enable animation

           4.delete all component in app.component html

           5.check 3rd party libraries working in project or not

           6.Generate components : ng g c component name
            -header
            -footer
            -sidebar
            -dashboard
                -card
                -piechart
                -chart
                -table
